# UI/UX 现代化优化详细工作计划 (Phase 3.6)

## 🎯 核心目标
基于 `ROADMAP.md` 中的 Phase 3.6，系统化地提升项目视觉体验，打造现代化、科技感的界面。

## 🔧 将调用的技能

### 1. **`frontend-design`** (前端设计专家技能)
**用途**：获取专业的前端界面设计指导，包括：
- 现代卡片设计模式（阴影、圆角、排版）
- 布局系统优化建议
- 交互细节（动画、反馈）
- 避免“通用AI审美”的设计原则

### 2. **`theme-factory`** (主题工厂技能)  
**用途**：确保设计一致性：
- 提取现有“石榴红+金”主题并现代化
- 定义CSS变量系统
- 生成配套的颜色/字体/间距规范

## 📋 详细工作步骤

### 阶段一：专业指导获取（技能调用）
1. **调用 `frontend-design` 技能**
   - 输入：当前 `CardEditor.vue` 组件截图/代码 + `ProjectBoard.vue` 界面
   - 请求：现代化设计建议（重点：卡片、侧边栏、整体布局）

2. **调用 `theme-factory` 技能**  
   - 输入：现有 `core/static/core/css/custom_admin.css` 中的主题变量
   - 请求：生成扩展的现代化主题系统

### 阶段二：设计系统建立
3. **创建 `docs/DESIGN_SYSTEM.md`**
   - 记录技能输出的设计建议
   - 定义颜色、字体、间距、阴影、圆角等设计Token
   - 建立组件库规范（按钮、卡片、输入框等）

4. **更新 Tailwind 配置**
   - 在 `frontend_dashboard/tailwind.config.js` 中集成新设计Token
   - 确保与后台主题的一致性

### 阶段三：组件级改造
5. **优化卡片样式 (`CardEditor.vue` + `ProjectBoard.vue`)**
   - 应用现代化阴影（`shadow-xl` → `shadow-2xl` 带模糊）
    -优化圆角（`rounded-xl` → `rounded-2xl`）
   - 改进内边距与字体层次
   - 添加微交互（Hover效果）

6. **优化后台侧边栏**
   - 分析 `backend/jazzmin_settings.py` 配置
   - 设计折叠动画与分组视觉优化
   - 保持“石榴粒粒”品牌色但现代化

7. **全局样式统一**
   - 审查并更新 `custom_admin.css`
   - 确保前后端设计语言一致

### 阶段四：验证与交付
8. **视觉回归测试**
   - 截图对比改造前后
   - 确保所有交互状态正常

9. **更新文档**
   - 在 `ROADMAP.md` 中标记 Phase 3.6 为进行中
   - 更新 `PROJECT_GUIDE.md` 中的设计系统说明

## 🎨 预期交付成果
1. **设计系统文档** (`docs/DESIGN_SYSTEM.md`)
2. **现代化的 Tailwind 配置**
3. **改造后的核心组件**：
   - `CardEditor.vue` (更现代、专业的编辑器)
   - `ProjectBoard.vue` (更美观的卡片网格)
   - `custom_admin.css` (升级后的主题)
4. **视觉对比报告** (改造前后截图)
