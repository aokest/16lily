# 产品功能说明书

## 1. 产品概述

**项目名称**：团队商机与业绩跟进系统 (Team Opportunity & Performance System)

**项目目标**：
本系统旨在为销售团队、市场部门及技术支持团队提供一个统一的协作平台，用于管理从商机挖掘、立项审批、市场活动推广到赛事运营的全生命周期业务数据。通过可视化的看板和自动化的流程，提高团队协作效率，实现数据的透明化和业绩的实时统计。

## 2. 用户角色与权限

系统主要服务于以下四类角色：

| 角色 | 典型岗位 | 核心权限 |
| :--- | :--- | :--- |
| **销售人员** | 销售经理/总监 | 录入商机、跟进客户、查看个人业绩、发起立项审批 |
| **春秋GAME** | 赛事运营/技术 | 录入赛事信息、审批赛事立项、维护赛事看板 |
| **集团市场部** | 市场经理 | 录入市场活动、审批活动预算/立项、维护活动看板 |
| **系统管理员** | 运维/管理层 | 用户管理、权限配置、AI模型配置、查看全盘数据 |

## 3. 核心功能模块

### 3.1 商机管理 (Opportunity Management)
销售团队的核心工作台，记录每一个销售机会的全过程。
- **商机录入**：记录客户名称、预计金额、负责销售等基础信息。
- **阶段推进**：管理商机从"接触"、"立项"、"招采"到"赢单"的全过程。
- **立项审批**：内置审批流，需经过销售部门负责人或管理员审批后方可正式立项。
- **业绩统计**：自动计算新签合同金额、毛利、回款等关键财务指标。

### 3.2 赛事运营管理 (Competition Ops)
专为"春秋GAME"团队设计，用于管理各类CTF比赛、实战演练等活动。
- **赛事库**：记录赛事名称、时间、地点、规模、赛制等详细信息。
- **导入功能**：支持从Excel批量导入历史赛事数据。
- **审批流**：赛事立项需经过GAME部门确认。
- **可视化看板**：提供卡片式看板（后台）和 **大屏展示**（前台），实时轮播赛事动态。

### 3.3 市场活动管理 (Market Activities)
专为"集团市场部"设计，用于管理展会、沙龙、峰会等市场推广活动。
- **活动日历**：记录活动的时间、地点、类型及规模。
- **审批流**：活动立项需经过市场部确认。
- **可视化看板**：提供活动状态看板和 **大屏展示**。

### 3.4 客户与联系人 (CRM Core)
- **客户档案**：建立企业客户数据库，记录行业、规模、法人等信息。
- **联系人管理**：记录客户方的关键决策人及联系方式。
- **数据关联**：商机自动关联客户，形成以客户为中心的360度视图。

### 3.5 智能辅助 (AI Integration)
- **AI配置中心**：后台支持配置 DeepSeek、Kimi、OpenAI 等大模型接口，支持 OCR 能力标记。
- **智能解析**：支持通过自然语言（如“下周三要交一份标书”）自动生成待办事项。
- **对话式工作汇报**：在工作报告页面提供 AI 聊天助手，通过对话方式自动梳理工作内容、生成摘要并提取下一步计划。

## 4. 交互体验

### 4.1 统一工作台 (Dashboard)
- **后台首页**：集成系统仪表盘，提供四大核心入口（商机、赛事、活动、战报）。
- **导航栏快捷入口**：顶部导航栏可直接跳转至各大看板。

### 4.2 数据大屏 (Big Screen)
专为办公区大屏幕设计的数据展示页面（访问端口：8080）：
- **赛事大屏**：轮播展示近期赛事、参赛规模统计图表。
- **活动大屏**：展示市场活动热力图、近期活动列表。
- **经营数据**：(规划中) 实时展示全公司的业绩完成情况。

## 5. 系统亮点

1.  **灵活的审批流**：摒弃复杂的财务审批，采用部门内闭环审批，更贴合业务敏捷性。
2.  **数据可视化**：不仅有传统的列表管理，更引入了Kanban（看板）和ECharts大屏，数据一目了然。
3.  **AI Ready**：底层已打通AI接口，为未来实现“对话式BI”和“智能助理”打下基础。
