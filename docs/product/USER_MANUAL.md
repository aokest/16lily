# 📘 石榴粒粒系统 - 用户操作手册 (User Manual)

> **适用版本**: v0.5.4-beta
> **最后更新**: 2025-12-12

欢迎使用 **石榴粒粒** 智能商机跟进系统。本手册根据不同角色为您提供操作指引。

---

## 🎭 角色与权限总览

系统主要包含以下几种角色，不同角色看到的菜单和功能有所不同：

| 角色 | 核心职责 | 核心权限 |
| :--- | :--- | :--- |
| **普通销售 (Sales)** | 录入商机、跟进客户、填写日报 | 仅能查看/编辑**自己**的商机和客户；无法查看他人数据。 |
| **销售经理 (Manager)** | 团队管理、商机审批、业绩考核 | 可查看**下属**的所有商机；拥有“批准立项”权限；可查看团队业绩报表。 |
| **售前/技术支持 (Pre-sales)** | 技术方案支持、POC测试 | 可被加入“项目组”查看特定商机；填写技术工作报告。 |
| **市场人员 (Marketing)** | 市场活动管理、线索分发 | 管理“市场活动”板块；录入线索并分配给销售。 |
| **系统管理员 (Admin)** | 系统配置、用户管理、AI设置 | 拥有所有权限；负责配置AI模型、部门结构和提示词。 |

## 🧑‍💼 普通销售操作指南

### 1. 快速录入商机 (AI 辅助)
这是您最常用的功能。
1.  点击左侧菜单 **“商机管理” -> “商机列表”**。
2.  点击右上角 **“增加 商机”**。
3.  **使用 AI 助手**: 在顶部的聊天框中，直接输入自然语言。
    *   *示例*: "新增一个商机，客户是九号电动车，预算15万，预计年底签约，我是付磊。"
    *   点击 **“生成”**，AI 会自动填好表单。
4.  **人工核对**: 检查 AI 填写的金额、日期是否准确。
5.  **保存**: 点击右下角“保存”。

### 2. 写日报
1.  点击左侧菜单 **“我的工作台” -> “工作报告”**。
2.  点击 **“增加 工作报告”**。
3.  **使用 AI 助手**: 输入今天的流水账。
    *   *示例*: "上午去拜访了张总，聊了二期需求，下午写了方案，明天打算去现场测试。"
    *   AI 会自动生成格式工整的 Markdown 日报，包含“今日工作”、“下一步计划”等。

### 3. 查看我的业绩
1.  点击左侧菜单 **“我的工作台” -> “业绩目标”**。
2.  您可以查看自己本季度、本年度的目标完成情况（百分比进度条）。

## 👨‍💼 销售经理操作指南

### 1. 审批商机立项
当销售录入新商机后，状态默认为“接触中”。如需立项：
1.  进入 **“商机列表”**。
2.  勾选需要审批的商机（前面的复选框）。
3.  在列表顶部的“动作”下拉框中，选择 **“批准立项 (仅销售/管理员)”**。
4.  点击 **“执行”**。商机状态将变更为“已立项”，并计入团队 Pipeline。

### 2. 查看团队报表
1.  进入 **“商机看板”** (如果有权限) 或查看大屏。
2.  利用列表页顶部的 **“过滤器”**，您可以按“部门”、“销售员”筛选查看下属的跟进记录。

## 🛠 系统管理员操作指南

### 1. AI 模型配置（新平台）
1.  进入 **“系统配置” -> “AI模型配置”**。
2.  您可以添加多个模型 (如 DeepSeek, Ollama, OpenAI)。
3.  **设为默认**: 在列表页直接点击开关，将其设为系统默认使用的模型。
4.  **测试连接**: 在“系统工具”页面点击“测试模型连接”，确保 API 通畅。
5.  **异常时兜底**: 模型不可用时，系统会在“智能执行面板”自动填表，不影响提交。

### 2. 修改 AI 提示词 (Prompt)
1.  进入 **“系统配置” -> “提示词模板”**。
2.  您可以分别编辑“商机录入”、“工作报告”等场景的提示词。
3.  系统会自动使用**最新且启用**的模板。建议修改前先复制一份旧模板作为备份。

### 3. 组织架构调整
1.  进入 **“系统配置” -> “部门管理”**。
2.  支持多级部门树状结构管理。
3.  点击右上角 **“查看组织架构图”** 可预览可视化树图。

## ❓ 常见问题 (FAQ)

**Q: 为什么我看不到“批准立项”按钮？**
A: 请确认您的角色是否为“销售经理”或“管理员”。普通销售无权审批。

**Q: AI 分析失败怎么办？**
A: 进入“系统工具”测试模型连接；若仍失败，系统会兜底填表，您可直接在“智能执行面板”确认后提交。管理员将进一步修复解析并更新提示词。

**Q: 忘记密码怎么办？**
A: 请联系系统管理员重置密码。
