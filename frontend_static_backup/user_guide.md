# 📘 十六军团战报系统 - 用户使用指南

## 1. 填写商机时的财务数据指南

在创建商机时，您可能会遇到一些不确定的财务字段。以下是填写建议：

### 💰 核心财务字段说明

| 字段名称 | 说明 | 填写建议 |
| :--- | :--- | :--- |
| **预计金额 (Estimated Amount)** | 这个项目大概能卖多少钱？ | **必填**。如果不确定，请填写一个**保守估计值**。后续可以在编辑中随时修改。 |
| **新签合同金额 (Signed Amount)** | 实际签合同的金额是多少？ | **赢单前可填0**。只有在商机状态变为“赢单”或接近签约时，才需要精确填写。 |
| **回款毛利 (Gross Profit)** | 扣除成本后赚多少？ | **选填**。如果不确定成本，可以先按公司平均毛利率估算（如 30%），或暂时填 0。 |
| **确认收入金额 (Revenue)** | 财务确认的收入是多少？ | **选填**。通常由财务或交付完成后填写，销售人员在初期可以不填或填 0。 |

### 💡 常见问题

**Q: 我还在接触阶段，根本不知道金额怎么办？**
A: 请根据客户的预算范围或同类项目的经验，填一个大概的数字（例如 500,000）。系统统计需要这个数字来计算漏斗规模。

**Q: 填错了可以改吗？**
A: 可以！随时进入“系统管理”后台，找到该商机进行修改。

## 2. 团队成员与工时配置 (新功能)

为了更准确地核算项目成本和人员投入，我们在后台增加了详细的团队成员配置功能。

### 👥 如何配置项目组成员？
1. 进入 **系统管理** -> **商机列表**。
2. 点击需要编辑的商机名称。
3. 在页面下方的 **“项目组成员”** 区域，点击 **“增加 项目组成员”**。

### 📝 填写项说明
- **系统用户**: 如果该成员是公司内部已注册用户，请直接选择。
- **成员姓名**: 如果是外部人员或未注册人员（如“其他部门人员”），请在此处直接填写姓名。
- **角色**: 选择该成员在项目中的角色（如：售前工程师、春秋GAME工程师、集团市场部研发人员等）。
- **责任分工**: 简要描述他在项目中负责什么（如：编写技术方案、负责POC测试）。
- **投入工时**: 预计或实际投入的人天数（1人天 = 8小时）。
- **进出时间**: 记录该成员进入和离开项目的具体日期。

## 3. 仪表盘功能说明

### 🏷️ 部门轮播与锁定
大屏左上角会显示当前的统计范围，支持以下部门/军团：
- **销售部**: 统揽全局销售数据。
- **春秋GAME**: 原POC部门，负责游戏化营销与POC测试。
- **集团市场部**: 负责市场活动与研发支持。
- **标准实践实验室**: 包含霄壤与伽玛实验室。
- **研发中心**: 负责产品研发与交付。

**操作方式**:
- **自动轮播**: 默认每 5 秒切换一次。
- **锁定查看**: 点击某个标签，会显示 🔒 图标，此时大屏将**锁定**在该部门的数据，不再自动切换。再次点击可解锁。

### 📊 统计指标解释
- **今日/本周/本月新增**: 统计的是**创建时间**在对应时间段内的商机数量。
- **已签约金额**: 统计的是所有状态为 **“赢单 (WON)”** 的商机中的 `新签合同金额` 总和。
- **商机池总额**: 统计的是所有商机的 `预计金额` 总和。

### 🔊 胜利音效
当有新的商机创建或状态更新时，大屏会播放一段激昂的 **“C大调胜利号角”** 音效，提醒团队又有好消息了！
